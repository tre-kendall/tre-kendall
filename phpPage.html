In my Database Programming Class I made this program that finds which two names are most alike in a list of names. This was early on into this year, so it's not an extremely advanced project. I do like this project because of my spacing. I believe I spaced the program out in a way that is easy to read and could be reused if necessary.

$DragonBallNames = array(
         "Goku",
         "Vegeta",
         "Freiza",
         "Broly",
         "Gohan",
         "Bulma",
         "Krillin",
         "Whis",
         "Beerus",
         "Chi-Chi",
         "Buu",
         "Uub");

$LevenshteinSmallest = 999999;
$SimilarTextLargest = 0;

for ($i=0; $i<11; ++$i) {
     for ($j=$i+1; $j<12; ++$j) {
          $LevenshteinValue = levenshtein($DragonBallNames[$i], $DragonBallNames[$j]);
          if ($LevenshteinValue < $LevenshteinSmallest) {
               $LevenshteinSmallest = $LevenshteinValue;
               $LevenshteinWord1 = $DragonBallNames[$i];
               $LevenshteinWord2 = $DragonBallNames[$j];
          }
          $SimilarTextValue =  similar_text($DragonBallNames[$i], $DragonBallNames[$j]);
          if ($SimilarTextValue > $SimilarTextLargest) {
               $SimilarTextLargest = $SimilarTextValue;
               $SimilarTextWord1 = $DragonBallNames[$i];
               $SimilarTextWord2 = $DragonBallNames[$j];
          }
     }
}

echo "< p>The levenshtein() function has determined that "$LevenshteinWord1"
      and "$LevenshteinWord2" are the most similar names.< /p>\n";
echo "< p>The similar_text() function has determined that "$SimilarTextWord1"
      and "$SimilarTextWord2" are the most similar names.< /p>\n";
